## Escape Test 
> It's easier to ask forgiveness <br> than it is to get permission <br> - Grace Hopper

 | Markdown | Render |
 |:----------------|:------------|
 |\**bold**, \_italic_|**bold**, _italic_|
 |\`code\`, \~strick~ | `code`,~~strike~~|
 |em\*\*\*bold***en|em**bold**en|

```go
import "fmt"

func main() {
	fmt.Println("Hello, World!")
}
```

````md
```go
import "fmt"

func main() {
	fmt.Println("Hello, World!")
}
```
````

```md
~~~md
**bold**, _italic_
`code`, ~strike~
em***bold***en
~~~
```
